- If notes is empty or does not start with `event:`, strategy_mode MUST be `evergreen_growth`.
- If notes starts with `event:`, strategy_mode MUST be either `event_active` or `event_transition`.
- `event_active`: the event is timely and may guide execution.
- `event_transition`: the event has passed or urgency is no longer valid; only bridge into relevant post-event execution if that remains compatible with Alysha strategy truth.
- Never invent an event when notes is empty.
